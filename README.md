# Breaking-Algorithms

A rigorous, timeline-driven collection of Core Data Structures and Algorithms implementations in C++.

## Project Roadmap
* **Week 1**: C++ STL fundamentals, Time & Space Complexity analysis, and Binary Search boundary-matching paradigms.
* **Week 2**: Greedy algorithms and local optimization techniques.
* **Week 3**: Overlapping subproblems, Memoization, and Tabulation using Dynamic Programming.
* **Week 4**: Advanced non-linear data structures covering Trees and Graph traversals.
